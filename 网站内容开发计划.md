### 1.3 设计语言（必须延续）
- **配色**：浅色 `--accent:#b45309` / 深色 `--accent:#f59e0b`；地球圈层色板 `--earth-*`；波形色板 `--wave-*`。所有新增颜色统一用 CSS 变量。

- **响应式**：≤768px 单栏 + 汉堡菜单；用已有的媒体查询断点，新增组件需自带移动端适配。

## 二、内容目标与原则

1. **科普性优先**：面向高中生，概念通俗、配可视化与类比，避免堆砌公式。
2. **与期刊定位绑定**：新增内容都要回扣《地震地质》的研究范畴（地震、构造、火山、地球物理），并在适当处引用期刊已发表方向。
3. **设计零新增成本**：尽量复用现有组件与样式，不引入新依赖、不破坏双主题与响应式。
4. **交互增强学习**：每个新增板块至少配置一种交互（点击切换 / 动画 / 时间轴 / 问答），与既有风格统一。

## 三、新增内容板块规划（6 个模块）

> 编号从「第八课」顺延。每个模块给出：定位、核心知识点、推荐呈现形式、所需数据/资源。

### 模块 A — 第八课：板块构造与地球动力学
- **核心知识点**：大陆漂移说 → 海底扩张 → 板块构造理论；三类板块边界（离散/汇聚/转换）；中国所处板块环境（欧亚板块与印度、太平洋板块相互作用）。
- **呈现形式**：
  - 顶部 `.section-body` 引入；
  - **交互 SVG**：世界板块边界示意图（可点击三类边界高亮 + 图例，复用 `#earth` 的「点击高亮」范式），或采用标签切换面板展示三类边界的成因与实例（如洋中脊、俯冲带、转换断层）。
  - `.card-grid` 列出 3 类边界 + 1 张「板块运动驱动力（地幔对流）」卡。
- **数据/资源**：边界类型说明文字；可选 SVG 简笔世界地图（用现有 viewBox 习惯绘制）。

### 模块 B — 第九课：矿物、岩石与岩石循环
- **核心知识点**：矿物 vs 岩石；三大岩类——岩浆岩（玄武岩/花岗岩）、沉积岩（砂岩/石灰岩）、变质岩（大理岩/片麻岩）；岩石循环（岩浆→喷出/侵入→风化剥蚀→沉积→变质→重熔）。
- **呈现形式**：
  - `.card-grid` 三类岩石各一卡（含 `.card-icon`、`.card-value` 举例、`.tag` 如「火成」）。
  - **交互 SVG / 流程图**：岩石循环三角图（点击任一岩类高亮其形成路径，复用 `layerData` 数据驱动范式）。
  - `.badge-section`：常见造岩矿物（石英、长石、云母、方解石）。
- **数据/资源**：三类岩石定义、代表岩、形成环境；循环关系数据。

### 模块 C — 第十课：地震学基础（震级·烈度·定位）
- **核心知识点**：震源、震中、震源深度；震级（里氏 / 矩震级 Mw）与烈度（感觉/破坏）的区别；地震定位原理（至少三台站到时差）；震级每差 1 级能量约差 30+ 倍。
- **呈现形式**：
  - `.card-grid` 拆解「震源/震中/震源深度」「震级 vs 烈度」「如何定位」三卡 + 1 张「能量尺度」卡。
  - **交互**：小动画或 SVG 示意「三台站交汇定位」（可选，二期）；或复用 `#knowledge` 的 Canvas 思路做「震级—能量」条形对比。
  - 与现有第三课（地震波）自然衔接，互相引用锚点。
- **数据/资源**：烈度表（中国地震烈度表）要点；能量—震级关系说明。

### 模块 D — 第十一课：地质年代与地球演化
- **核心知识点**：相对年代 vs 绝对年代；地质年代表（冥古/太古/元古/显生宙；代—纪—世）；新生代与人类文明；用放射性同位素定年（呼应第六课测年法）。
- **呈现形式**：
  - **交互时间轴**：复刻 `.timeline` 组件，但改为横向/纵向「地质年代尺」——点击「纪」显示该时期代表事件（如显生宙的生物大爆发、恐龙灭绝、人类出现）。
  - `.badge-section`：关键界线年龄（寒武纪底 ~5.41 亿年等）。
- **数据/资源**：年代表数据数组（宙/代/纪/起止年龄/代表事件），用于 JS 渲染。

### 模块 E — 第十二课：地球物理勘探方法
- **核心知识点**：在第六课（InSAR/GPS/探槽/测年）基础上，补充**地球物理探测四大家**——重力勘探、磁法勘探、电法勘探、地震勘探（反射/折射）；说明各自探测对象（密度/磁性/电性/波速差异）。
- **呈现形式**：
  - `.card-grid` 四类方法各一卡（含 `.card-icon`、`.card-value` 探测物理量、`.tag` 应用）。
  - 与第六课合并或紧邻，避免重复；可在第六课后插入「地球物理方法总览」作为子板块。
- **数据/资源**：四类方法原理一句话说明 + 典型应用（找矿、找水、油气、工程勘察）。

### 模块 F — 第十三课：地表过程与地质灾害
- **核心知识点**：外动力地质作用——风化、侵蚀、搬运、沉积；典型地质灾害——滑坡、泥石流、地面沉降、地裂缝（呼应地震次生灾害）；海啸成因（呼应第七课日本案例）。
- **呈现形式**：
  - `.card-grid` 外动力四作用 + 主要灾害类型卡。
  - `.badge-section`：防灾要点（临震避险、滑坡前兆等科普提示）。
- **数据/资源**：各灾害定义与诱因；与既有「汶川滑坡/堰塞湖」内容呼应。

> 可选（二期）：模块 G — 地磁场与古地磁（极性倒转、海底磁条带作为海底扩张证据），作为板块构造的延伸证据。

## 八、风险与备注
- **内容准确性**：地质年代数字、震级—能量关系、岩石分类等需以权威资料（如《地震地质》已发表综述、教科书）为据，避免口误。
- **篇幅控制**：单页内容不宜无限膨胀；若内容过多，建议二期拆分为多页或折叠式「阅读更多」。
- **SVG 资源**：板块边界图、岩石循环图建议用矢量手绘（保持扁平学术风），不使用外部图片以保证加载与主题色统一。
